# 景点图标说明

本目录存放地图上显示的景点标记图标。

## 图标命名规范

- `spot-5a.png` - 5A级景区图标 (32x32px)
- `spot-4a.png` - 4A级景区图标 (32x32px)
- `spot-3a.png` - 3A级景区图标 (32x32px)
- `spot-2a.png` - 2A级景区图标 (32x32px)
- `spot-default.png` - 默认景区图标 (32x32px)

## 图标设计建议

1. **尺寸**: 32x32px 或 64x64px (@2x)
2. **格式**: PNG (支持透明背景)
3. **颜色**: 
   - 5A: 红色系 (#ff6b6b)
   - 4A: 青色系 (#4ecdc4)
   - 3A: 蓝色系 (#45b7d1)
   - 2A: 绿色系 (#96ceb4)
   - 默认: 灰色系 (#95a5a6)

## 临时方案

如果暂时没有图标文件，系统会使用地图标记的 callout (文字气泡) 显示景点信息，不影响功能使用。

## 获取图标

你可以：
1. 自己设计图标
2. 使用在线图标库 (如 iconfont.cn)
3. 使用emoji作为临时方案（已在代码中配置）
